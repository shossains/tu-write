Referencing using footnote style

Characteristics of the footnote style:

  • This style is often used in medicine and law.
  • In-text-citation: use a superscripted number to refer to background literature.
  • In a footnote at the bottom of the page, the information about the source is mentioned.
  • Usually, the first time that a specific publication is used, the whole title description of that source is provided in the footnote. If the same publication is used again, a short reference to the publication is given.
  • Example: Chicago style.
