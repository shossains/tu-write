Enumerations and lists

A special kind of paragraph is the enumeration, which often has the format of a list (numbered or bulleted).

Example of an enumeration

This enumeration has three elements. Note that all elements have the same structure. Note the colon (dubbele punt) after the announcing sentence and the semicolons (puntkomma’s) between the elements.

Certain functionalities need to be implemented to defeat basic units. We defined requirements as a Must have if these were necessary to win the game. Our Must haves are:

  • being able to gather basic resources;
  • being able to scout the enemy base;
  • building the right buildings at the right time;
  • being able to attack a building.

Rules for lists

  • Announce the list. Use a heading or an introductory or announcing sentence.
  • Announce the number of elements.
  • Bulleted list: add a minimum of two and a maximum of six elements. A numbered list can be longer.
  • Make sure that all elements have the same sentence structure.
  • Use the same format throughout the report.
