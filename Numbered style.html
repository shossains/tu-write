Referencing using numbered style

Characteristics of the numbered style:

  • This style is often used in the sciences.
  • In-text citations: use a number between square brackets to refer to a reference. Add the page number(s) when quoting a source.
  • In-text citations are usually numbered in order of first appearance in the text.
  • The list of references at the end of the text is ordered by number.
  • As a reader, if you are interested in the exact publication, you simply look up the number of the in-text citation in the reference list at the end of the text.
  • Examples of this reference style: IEEE, AIAA and ACS.
